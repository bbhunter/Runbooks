# ASNmap

**Description:** A entry to describe how to use ASNmap to collect ASN ranges

**Requirements:** go, asnmap

## Running ASNMap

```
asnmap -org <org_name>
```

Lookup ASN IP ranges by organization name

```
asnmap -a <ASN_ID> -silent
```

Lookup ASN IP ranges by ASN ID
