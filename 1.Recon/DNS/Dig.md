# Dig

**Description:** This entry describes how to use Dig to lookup domains

**Requirements:** dig

## Running Dig

```
dig +short <hostname>
```

Forward lookup on hostname

```
dig +short -x <ip_address>
```

Reverse Lookup on ip address
