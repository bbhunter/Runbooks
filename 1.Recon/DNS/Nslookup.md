# Nslookup

**Description:** This entry describes how to use Nslookup to lookup domains

**Requirements:** nslookup

## Running Nslookup

```
nslookup
>set type=cname
>domain.com
```

Lookup different record types for domain name (e.g. a,mx,ns)
